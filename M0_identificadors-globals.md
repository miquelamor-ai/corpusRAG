# 1. CONTINGUT ESPECÍFIC DEL TIPUS "MARC"

## Definició i principis

Els Identificadors Globals de Jesuïtes Educació (FJE) són els trets identitaris i valors fonamentals que defineixen la proposta educativa dels centres de la Companyia de Jesús a Catalunya. Aquests identificadors s'arrelen profundament en l'espiritualitat ignasiana i en una tradició educativa de gairebé 500 anys, adaptada als reptes i necessitats del segle XXI. La seva finalitat última és la formació integral de la persona, entesa com un procés de creixement humà i espiritual que busca el "major bé" (Magis) per a l'individu i la societat.

El cor d'aquesta identitat és la convicció d'“*trobar Déu en totes les coses*” (DOC 1, DOC 55), que es tradueix en una actitud de “*contemplatiu en l'acció*” (DOC 1, DOC 63). Això significa que la fe no és un àmbit separat de la vida, sinó que impregna cada aspecte de l'existència i de l'aprenentatge. L'educació jesuïta busca anar més enllà de la mera transmissió de coneixements, promovent una “*formació integral que desenvolupi el potencial de la persona al servei de la comunitat, del bé comú, d'un món millor per a totes les persones*” (DOC 43, DOC 79, DOC 109).

Aquesta missió es concreta en el servei de la fe i la promoció de la justícia, considerant aquesta última com una “*exigència absoluta*” (DOC 68, DOC 94). Això implica un compromís actiu amb la transformació social i la construcció d'un món més equitatiu i reconciliat (DOC 34, DOC 68, DOC 88). La Companyia de Jesús, a través de les seves Congregacions Generals (CG32, CG36), ha subratllat la importància de la reconciliació amb Déu, amb la humanitat i amb la creació (DOC 68, DOC 88, DOC 103).

Un element central és el concepte de "cura personalis" (DOC 54, DOC 93), que posa l'accent en l'atenció personalitzada a cada alumne, respectant la seva singularitat, ritmes i necessitats (DOC 11, DOC 79). Aquesta atenció es dirigeix a acompanyar l'alumnat en la construcció del seu “*projecte vital i vocacional*” (DOC 10, DOC 11, DOC 12, DOC 13, DOC 16, DOC 17, DOC 74, DOC 76), ajudant-los a descobrir el sentit de la seva vida i la seva aportació al món.

Els Identificadors Globals de FJE es materialitzen en el perfil de l'alumnat que es vol formar: persones “*conscients, competents, compassives i compromeses*” (les 4 Cs) (DOC 11, DOC 14, DOC 15, DOC 17, DOC 43, DOC 85, DOC 109, DOC 113). Aquestes característiques, inspirades en la visió de Pedro Arrupe de formar “*homes i dones per als altres i amb els altres*” (DOC 14, DOC 46, DOC 53, DOC 109, DOC 113), guien totes les propostes pedagògiques i metodològiques.

Finalment, els centres FJE es comprometen amb la “*ciutadania global*” i la “*cura de la Casa Comuna*” (DOC 13, DOC 14, DOC 15, DOC 20, DOC 25, DOC 26, DOC 27, DOC 29, DOC 86, DOC 94, DOC 96, DOC 97, DOC 102, DOC 103, DOC 113), responent als reptes socioambientals del nostre temps i promovent una “*globalització de la solidaritat, la cooperació i la reconciliació*” (DOC 86, DOC 96). El discerniment (DOC 3, DOC 4, DOC 29, DOC 36, DOC 61, DOC 63, DOC 85, DOC 87, DOC 88, DOC 89, DOC 102, DOC 110, DOC 113) i el treball en xarxa (DOC 36, DOC 37, DOC 47, DOC 83, DOC 86, DOC 91, DOC 95, DOC 96, DOC 103, DOC 111) són modes de procedir essencials per a la realització d'aquesta missió.

## Autors i evidència clau

Els Identificadors Globals FJE s'inspiren en una rica tradició de pensament i pràctica educativa de la Companyia de Jesús, amb figures i documents clau:

*   **Sant Ignasi de Loiola (1491-1556):** És el fundador del carisma ignasià. Els seus *Exercicis Espirituals* (DOC 1, DOC 4, DOC 5, DOC 6, DOC 29, DOC 36, DOC 55, DOC 58, DOC 61, DOC 62, DOC 63, DOC 65, DOC 67, DOC 85, DOC 87, DOC 104, DOC 110, DOC 113) són la font de la pedagogia ignasiana, centrada en el discerniment, la reflexió i l'acció. Les *Constitucions de la Companyia de Jesús* (DOC 7, DOC 8) estableixen la missió fundacional de la Companyia.
*   **Pedro Arrupe, S.J. (Pare General 1965-1983):** Va articular el concepte de "Homes i dones per als altres" (DOC 14, DOC 46, DOC 53, DOC 109, DOC 113) com a finalitat de l'educació jesuïta, promovent una fe que es fa justícia (DOC 45, DOC 77, DOC 111). La CG32, sota el seu lideratge, va definir la missió de la Companyia com el "servei de la fe, del que la promoció de la justícia constitueix una exigència absoluta" (DOC 68, DOC 94, DOC 111).
*   **Peter-Hans Kolvenbach, S.J. (Pare General 1983-2008):** Va consolidar la visió educativa amb documents com les *Característiques de l'Educació de la Companyia de Jesús* (DOC 32, DOC 73, DOC 77, DOC 105, DOC 106, DOC 111) i la *Pedagogia Ignasiana: un plantejament pràctic* (DOC 77, DOC 99, DOC 111, DOC 112). Va introduir les "quatre Cs" (Competents, Conscients, Compassius, Compromisos) com a síntesi de l'excel·lència humana (DOC 43, DOC 109, DOC 113) i va desenvolupar el Paradigma Ledesma-Kolvenbach (utilitas, iustitia, humanitas, fides) (DOC 108).
*   **Adolfo Nicolás, S.J. (Pare General 2008-2016):** Va posar l'accent en la "profunditat i universalitat" de l'educació jesuïta (DOC 33, DOC 37, DOC 39, DOC 41, DOC 51, DOC 111) i en el creixement i transformació de la persona (DOC 40, DOC 41, DOC 51).
*   **Arturo Sosa, S.J. (Pare General 2016-actualitat):** Ha impulsat les *Preferències Apostòliques Universals (PAU) 2019-2029* (DOC 14, DOC 15, DOC 28, DOC 29, DOC 30, DOC 68, DOC 87, DOC 88, DOC 89, DOC 90, DOC 96, DOC 97, DOC 102, DOC 103, DOC 111, DOC 113), que guien la missió de la Companyia en l'actualitat: mostrar el camí cap a Déu, caminar amb els exclosos, acompanyar els joves i cuidar la Casa Comuna. Ha emfatitzat la reconciliació i la ciutadania global (DOC 34, DOC 35, DOC 49, DOC 52, DOC 53, DOC 68, DOC 88, DOC 89, DOC 91, DOC 94, DOC 96, DOC 103, DOC 111, DOC 113).
*   **Magisteri Pontifici:** Les encícliques del Papa Francesc, com *Laudato Si'* (ecologia integral) (DOC 25, DOC 26, DOC 97) i *Fratelli Tutti* (fraternitat i amistat social) (DOC 21, DOC 23), així com el *Pacte Educatiu Global* (DOC 20, DOC 27), proporcionen un marc teològic i ètic fonamental que s'alinea amb la missió ignasiana.

**Tensions entre autors/documents:**

Una tensió rellevant, assenyalada al *Diccionario de Espiritualidad Ignaciana* (DOC 58), és la que pot sorgir entre la **espiritualitat ignasiana tradicional**, que se centra en la transformació personal a través de l'ordenació de les "afeccions desordenades", i la **missió actual de promoció de la justícia**, que implica una transformació social i pot generar conflictes amb el sistema establert. El repte és integrar ambdós projectes (el religiós i el social) de manera que la formació personal condueixi al compromís social actiu, sense que un dilueixi o invisibilitzi l'altre.

## Exemples concrets d'aplicació a l'aula

Els Identificadors Globals FJE es concreten en la pràctica educativa diària a totes les etapes:

*   **Educació Infantil (MOPI) i Primària Inicial (PIN):**
    *   **Projecte vital i vocacional des de la primera infància:** Acompanyar el creixement integral de l'infant, potenciant la seva curiositat, autonomia, creativitat, empatia i responsabilitat (DOC 11, DOC 16). Es fomenta que l'infant es faci preguntes i busqui respostes, desenvolupant la seva capacitat d'aprendre i de relacionar-se amb els altres i l'entorn.
    *   **Aprenentatge globalitzat i significatiu:** A través de "nodes" temàtics (MOPI) o "caixes de recerca" (PIN), es promou la investigació i la reflexió sobre el món que els envolta. L'aprenentatge de les llengües es concep com una competència contínua, personalitzada i amb sentit en situacions comunicatives autèntiques (DOC 16, DOC 18).
    *   **Exemple:** En un projecte sobre "L'aigua" a P5, els infants investiguen d'on ve l'aigua, com es fa servir, i quina importància té per a la vida. A través de jocs, experiments i sortides, descobreixen la necessitat de cuidar aquest recurs, connectant amb la cura de la Casa Comuna des de petits. Es fomenta la conversa i l'expressió de les seves idees i emocions, desenvolupant la seva consciència i empatia.

*   **Educació Secundària Obligatòria (NEI i TQE):**
    *   **Aprenentatge Basat en Projectes (ABP) per a reptes reals:** Es proposen situacions-problema que connecten amb els interessos de l'alumnat i fomenten el pensament crític, el treball en equip i la iniciativa (DOC 17). L'objectiu és capacitar per al compromís social des de la pròpia llibertat.
    *   **Projecte vital i vocacional per a la presa de decisions amb sentit:** A través de l'acció tutorial, projectes d'àmbit i itineraris optatius, s'acompanya l'alumnat a reflexionar sobre la seva vida, a construir una cultura relacional des de la fraternitat i a buscar el sentit transcendent de l'existència (DOC 12, DOC 17).
    *   **Exemple:** Un projecte de 2n d'ESO sobre "La pobresa al meu barri" implica que els alumnes investiguin les causes i conseqüències de la pobresa local, entrevistin entitats socials, i proposin accions concretes de millora. Aquesta experiència els ajuda a desenvolupar les 4 Cs: ser conscients de la realitat, competents en la recerca i anàlisi, compassius amb els afectats i compromesos amb la justícia social, connectant amb la PAU de "caminar amb els exclosos".

*   **Batxillerat i Formació Professional (FP):**
    *   **Formació de persones amb consciència ètica i compromís amb la justícia:** Es promou la reflexió crítica sobre els desafiaments globals i la capacitat de "transformar-se per transformar" el món (DOC 10, DOC 12). L'alumnat és acompanyat per assolir el protagonisme en el seu aprenentatge i integrar-se amb èxit al món laboral amb una mirada de servei.
    *   **Orientació del projecte vital i vocacional:** S'ofereix un acompanyament personal i acadèmic per ajudar els joves a orientar els seus estudis i el seu futur professional des dels seus interessos, habilitats i valors, preparant-los per a una formació permanent al llarg de la vida (DOC 10, DOC 12, DOC 13).
    *   **Exemple:** Un estudiant de Batxillerat que participa en un programa de voluntariat internacional a l'estiu, reflexiona sobre la seva experiència en un diari personal i la comparteix amb el grup. Aquesta activitat fomenta la consciència global, la compassió i el compromís, i l'ajuda a discernir com la seva futura professió pot ser un instrument de servei i transformació social, alineant-se amb la PAU d'"acompanyar els joves en la creació d'un futur esperançador".

## Errors comuns — què NO fer

Per mantenir la fidelitat als Identificadors Globals FJE, és important evitar certes pràctiques que podrien desvirtuar la missió:

1.  **Reduir la formació en la fe a una simple assignatura o catequesi desconnectada de la vida:** La fe, en l'educació jesuïta, no és només un contingut curricular a impartir, sinó el fonament explícit i transversal que impregna tota la cultura del centre (DOC 89, DOC 92, DOC 102). Si es limita a un espai o horari concret, es perd la seva capacitat de transformar la perspectiva de l'alumne, d'anar més enllà dels coneixements operacionals i d'encarnar-se en la comunitat. L'objectiu és que la fe sigui una perspectiva personal que ajudi a "glorificar Déu en totes les coses" (DOC 89), no una mera transmissió doctrinal.
2.  **Limitar la "cura personalis" a l'atenció individualitzada exclusivament acadèmica o de resolució de problemes:** La "cura personalis" és una atenció integral a totes les dimensions de la persona (física, intel·lectual, social, espiritual, artística, tècnica) per al seu creixement i transformació (DOC 41, DOC 51, DOC 54, DOC 79, DOC 93, DOC 109). Si es redueix a un suport merament acadèmic o a la intervenció davant dificultats, es perd la visió holística de l'acompanyament del projecte vital i vocacional de l'alumne, que busca potenciar les seves fortaleses i el seu desenvolupament ple.
3.  **Entendre la innovació pedagògica com una fi en si mateixa, desconnectada de la missió i els valors ignasians:** La innovació pedagògica a FJE ha d'estar sempre al servei de la missió de formar "homes i dones per als altres" i per a una ciutadania global (DOC 13, DOC 14, DOC 15, DOC 82). Si s'adopten noves metodologies sense una reflexió profunda sobre com aquestes contribueixen al desenvolupament de les 4 Cs (conscients, competents, compassius, compromesos) i al compromís amb la justícia i la reconciliació, es corre el risc de caure en la superficialitat o en la simple "moda" pedagògica, sense un impacte real en la formació integral de l'alumnat.
4.  **Ignorar la dimensió social i global de l'educació, centrant-se exclusivament en l'àmbit local o individual:** L'educació jesuïta no pot aïllar-se de la realitat social i dels reptes globals (DOC 14, DOC 35, DOC 82, DOC 94, DOC 96). Si la pràctica educativa no connecta l'alumnat amb els problemes del món (injustícia, desigualtat, crisi ecològica) i no els impulsa a actuar com a ciutadans globals, es desaprofita el potencial transformador de l'educació. L'objectiu no és només formar bons professionals, sinó persones compromeses amb la millora del món a favor del bé comú (DOC 13, DOC 15).

## Matissos i excepcions

Els Identificadors Globals FJE són un marc sòlid, però la seva aplicació requereix matisos i una comprensió de les possibles tensions:

*   **Integració de la transformació personal i social:** El *Diccionario de Espiritualidad Ignaciana* (DOC 58) assenyala que la espiritualitat ignasiana tradicional, centrada en la transformació de la persona (ordenació de les afeccions desordenades), pot no connectar de forma automàtica amb la ineludible necessitat de defensar la justícia en el món, la qual cosa pot generar conflictes amb el sistema establert. El repte no és triar entre una o altra, sinó integrar ambdós projectes (religiós i social) de manera que la formació personal condueixi naturalment al compromís social actiu, sense que la "tranquil·litat de consciència" apacigui l'esperit i impedeixi l'acció transformadora.
*   **Educació en la fe en contextos seculars i multiculturals:** Les escoles jesuïtes han de ser "catòliques i oferir formació profunda en la fe en diàleg amb altres religions i visions del món" (DOC 89, DOC 90, DOC 92, DOC 102). Això implica que la identitat catòlica no ha de ser excloent ni imposada, sinó proposada des del testimoni i el diàleg, reconeixent la diversitat religiosa i cultural de l'alumnat i la societat. El repte és ser "ponts apostòlics entre la bellesa de la diversitat creada per Déu i la nostra fe" (DOC 90).
*   **Equilibri entre l'excel·lència acadèmica i la formació humana integral:** L'excel·lència a FJE no es limita als resultats acadèmics, sinó que busca el "màxim desenvolupament dels dons i capacitats amb què cada persona ha estat dotada... per al millor servei dels altres" (DOC 109, DOC 113). El perill és que la pressió per obtenir bones notes o l'accés a estudis superiors desvirtuï la formació en les 4 Cs, el desenvolupament de la consciència crítica, la compassió i el compromís social. Cal mantenir un equilibri que valori ambdues dimensions de l'excel·lència.
*   **La universalitat del carisma ignasià versus la diversitat dels contextos locals:** La Companyia de Jesús aspira a "actuar com un cos universal amb una missió universal", però reconeix la "radical diversitat de les nostres situacions" (DOC 86, DOC 96, DOC 102). Això significa que els Identificadors Globals no són un manual rígid, sinó un marc d'inspiració que ha de ser adaptat i encarnat creativament en cada realitat local, respectant les cultures, les necessitats socioeconòmiques i les ideologies de cada territori (DOC 9, DOC 70).

# 3. DETECCIÓ (Variables de Context)

*   **Senyals del docent:**
    *   "Em pregunto quin és el propòsit últim de la meva feina més enllà del currículum acadèmic."
    *   "Busco maneres d'integrar valors ètics, socials i espirituals en les meves classes diàries."
    *   "Vull que els meus alumnes desenvolupin un sentit crític i un compromís actiu amb el món."
    *   "Com puc anar més enllà de la transmissió de continguts i formar persones de manera integral?"
    *   "Em sento interpel·lat pels reptes globals (canvi climàtic, desigualtats) i vull que la meva pràctica educativa hi respongui."
    *   "Necessito eines per ajudar els meus alumnes a reflexionar sobre el seu futur i el seu paper en la societat."
*   **Senyals de l'alumne:**
    *   L'alumne expressa inquietuds sobre el sentit de la vida, el seu futur o la seva vocació personal.
    *   Mostra interès per problemes socials o mediambientals i manifesta el desig d'actuar o contribuir a la seva solució.
    *   Busca connexions significatives entre el que aprèn a l'escola i la seva vida personal, els seus valors o el món que l'envolta.
*   **Senyals de context:**
    *   Inici d'un nou projecte educatiu de centre o revisió del Projecte Educatiu de Centre (PEC).
    *   Implementació de noves metodologies actives que busquen un aprenentatge més profund i competencial.
    *   Debats o conflictes a l'aula o a la comunitat educativa sobre valors, ètica, justícia social o sostenibilitat.
    *   Demanda de les famílies per una educació que transcendeixi l'acadèmic i formi en valors humans.
*   **Anti-senyals:**
    *   El docent només busca solucions ràpides i superficials per complir amb el currículum sense una reflexió de fons sobre el propòsit educatiu.
    *   L'alumne mostra una actitud totalment apàtica i desinteressada per qualsevol aspecte que vagi més enllà de l'aprovat o l'interès immediat.
    *   El context escolar està completament centrat en resultats acadèmics estandarditzats i la competitivitat, sense espai per a la formació integral o el compromís social.
    *   La comunitat educativa percep la "identitat" com un element merament formal o decoratiu, sense implicació real en la pràctica diària.

# 4. HEURÍSTIQUES I RAONAMENT PER A L'AGENT

*   **Principi general:** L'educació FJE busca la formació integral de persones que, des d'una profunda relació amb Déu i amb elles mateixes, siguin competents, conscients, compassives i compromeses amb la justícia i la cura de la Casa Comuna, actuant com a ciutadans globals.

*   **Heurístiques per a l'Agent DOCENT:**

    *   **Heurística 1: Integració de les 4 Cs en el disseny d'activitats.**
        *   **Quan aplica:** En la planificació de qualsevol unitat didàctica, projecte o activitat a l'aula, des de qualsevol àrea curricular.
        *   **Fonament:** Els Identificadors Globals FJE emfatitzen la formació de persones "conscients, competents, compassives i compromeses" (DOC 11, DOC 14, DOC 15, DOC 17, DOC 43, DOC 85, DOC 109, DOC 113). Aquestes quatre dimensions no són aïllades, sinó que es reforcen mútuament per a una excel·lència humana integral.
        *   **Exemple de raonament:** Si un docent està dissenyant un projecte sobre el canvi climàtic per a alumnes d'ESO, l'agent podria suggerir que, més enllà de la recerca de dades científiques (competència) i la comprensió de les causes i efectes (consciència), s'incloguin activitats que fomentin l'empatia cap a les comunitats més vulnerables afectades pel canvi climàtic (compassió) i propostes d'acció local o global per mitigar-lo o adaptar-s'hi (compromís). Per exemple, es podria demanar als alumnes que investiguin l'impacte en un país en desenvolupament, que creïn un manifest demanant mesures concretes a les autoritats locals o que organitzin una campanya de sensibilització a l'escola. Això assegura que l'aprenentatge no sigui només cognitiu, sinó que mobilitzi la persona en la seva totalitat, alineant-se amb la missió de formar ciutadans globals i la cura de la Casa Comuna.

    *   **Heurística 2: Foment del Projecte Vital i Vocacional (PVV).**
        *   **Quan aplica:** En moments d'orientació acadèmica i personal, en tutories, o en la introducció de nous projectes que permetin als alumnes reflexionar sobre el seu futur i el seu paper en el món.
        *   **Fonament:** FJE acompanya l'alumnat en la construcció del seu PVV, entès com la recerca de sentit i la donació de la pròpia vida (DOC 10, DOC 11, DOC 12, DOC 13, DOC 16, DOC 17, DOC 74, DOC 76).
        *   **Exemple de raonament:** Si un docent de Batxillerat està ajudant els seus alumnes a escollir estudis superiors o una carrera professional, l'agent podria proposar activitats que transcendeixin la simple elecció basada en interessos o notes. Per exemple, podria suggerir que els alumnes investiguin com la professió que els interessa pot contribuir al bé comú, a la resolució de problemes socials o a la promoció de la justícia, o que reflexionin sobre quins valors personals volen projectar en el seu futur professional. Això els ajuda a connectar les seves aspiracions acadèmiques amb una visió més àmplia de la seva vocació i propòsit vital, fomentant una elecció més conscient, lliure i compromesa amb la societat, un dels pilars de la identitat FJE.

    *   **Heurística 3: Promoció del discerniment en la presa de decisions.**
        *   **Quan aplica:** Quan els alumnes s'enfronten a decisions importants, ja siguin acadèmiques, personals o grupals, o davant dilemes ètics.
        *   **Fonament:** El discerniment ignasià és una pràctica fonamental per a la presa de decisions lliures i conscients, buscant la voluntat de Déu o el "major bé" (DOC 3, DOC 4, DOC 29, DOC 36, DOC 61, DOC 63, DOC 85, DOC 87, DOC 88, DOC 89, DOC 102, DOC 110, DOC 113).
        *   **Exemple de raonament:** Davant un conflicte en un treball en grup a l'aula de Secundària, l'agent podria guiar el docent a facilitar un procés de discerniment comunitari. Això implicaria no només buscar una solució immediata, sinó animar els alumnes a expressar les seves "mocions" (sentiments, pensaments, intuïcions), a escoltar activament les perspectives dels altres, a ponderar les conseqüències de les diferents opcions i a buscar la solució que generi més "pau" i "major bé" per al grup i el projecte. Aquest procés els ensenya a anar més enllà de la reacció impulsiva i a prendre decisions més reflexives, ètiques i col·laboratives, desenvolupant la seva capacitat de lideratge i el sentit de responsabilitat compartida.

    *   **Heurística 4: Integració de l'ecologia integral i la ciutadania global.**
        *   **Quan aplica:** En el disseny de projectes interdisciplinaris, activitats de sensibilització o en la gestió del centre.
        *   **Fonament:** La cura de la Casa Comuna i la ciutadania global són pilars de la missió jesuïta, que responen a la "complexa crisi socio-ambiental" i promouen la "globalització de la solidaritat" (DOC 13, DOC 14, DOC 15, DOC 20, DOC 25, DOC 26, DOC 27, DOC 29, DOC 86, DOC 94, DOC 96, DOC 97, DOC 102, DOC 103, DOC 113).

A més, es podria connectar amb iniciatives similars en altres escoles jesuïtes del món a través de la xarxa Educate Magis, per fomentar una visió de ciutadania global i de responsabilitat compartida en la cura de la creació. Això implica que, abans de saltar a les solucions, cada membre comparteixi la seva perspectiva des de l'experiència personal i els seus sentiments ("parlar des del cor"), escoltant amb "reverència i acollida" les aportacions dels altres (DOC 5). Es tracta de crear un ambient de confiança on es puguin expressar les "mocions" interiors i discernir junts la millor acció, reconeixent la presència de l'Esperit en el diàleg.
